## How To Use :closed_lock_with_key:

```bash
# Clone this repository
$ git clone git@github.com:mux-mux/food.git

# Go into the repository
$ cd food

# Install dependencies
$ npm install

# Open browser, listen to scss js changes
$ npm run dev
```
